# El Loco Enterprise

Website for El Loco Enterprise Pte. Ltd., a wholesale and retail snack shop in
Sembawang, Singapore. Three pages: home, product catalog, and about/contact.
There is no online checkout — the site points customers to phone and WhatsApp.

Built with React, TypeScript, Vite, Tailwind CSS, and React Router.

## Development

```sh
npm install
npm run dev      # local dev server
npm run build    # type-check and build to dist/
npm run preview  # serve the production build
npm run lint     # oxlint
```

Product and contact details live in `src/data/products.ts`.
